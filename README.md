## 👨‍💻 About Me

I am **Arka Talukder**, a cybersecurity researcher and practitioner, currently completing my **MSc in Cyber Security at the University of the West of Scotland (UWS), London Campus** (2025–2026), supervised by Dr Raja Ujjan.

My dissertation — *"Explainable Dynamic Graph Neural Network SIEM for Software-Defined IoT using Edge AI and Federated Learning"* — is a CPU-oriented prototype SIEM built around a Dynamic GNN (GATConv + GRU) with Federated Learning (Flower FedAvg) and Explainability (Captum Integrated Gradients), serving SOC-ready alerts via FastAPI.

My background blends physics-driven analytical thinking (BSc Physics, SUST) with hands-on security operations experience as a **Cyber Security Engineer at StepUp Technology Ltd.** (Jul 2024 – May 2025), where I helped secure a production network of 2,000+ users. I also founded **BDLearner.com** (2019), which taught me to build, maintain, and secure web infrastructure end-to-end.

🎯 **Open to:** SOC Analyst, Security Engineer, AI/ML Security Researcher, Malware Analyst roles — on-site or hybrid in Bangladesh, UK, or remote.
